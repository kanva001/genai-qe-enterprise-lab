# Definition of Done (DoD)

A story is Done when:
- Code is committed with tests (where applicable)
- Evidence is produced (report/log/screenshot)
- Documentation updated (docs/confluence)
- CI is green (where applicable)

For GenAI outputs:
- Prompt template is versioned
- Output follows schema (JSON/YAML)
- Human review note is captured in docs/confluence/Defect_Triage_Log.md or a PR description
